== Description ==
Add custom content to post and/or page (top/bottom). You may use text, HTML, Shortcodes and JavaScript. Simple, but work...

== Installation ==
1. Upload `pryc-wp-add-custom-content-to-bottom-of-post` dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
